Method MakeCommand
Creates a command instance based on the given command type.
Syntax
C#
    public override ICommand MakeCommand(string commandType)
Parameters
commandType
A string representing the type of command to create. Supported types include "triangle", "write", "int", "real", "rect", "reset", "while", and "end".
Return Value
An instance of a class implementing the ICommand interface corresponding to the specified command type.
Exceptions
Exception Condition
ArgumentException Thrown when the commandType is null, empty, or consists only of whitespace.
Remarks
This method normalizes the commandType to lowercase and trims any leading or trailing whitespace. If the command type is not recognized, it falls back to the base implementation of MakeCommand.
