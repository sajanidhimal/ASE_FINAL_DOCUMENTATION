Class AppFor
Represents a custom implementation of the For class for creating and executing a for loop. Allows for a loop with customisable start, end, and step values, along with a block of code to execute in each iteration.
Syntax
C#
    public class AppFor : For

Documentation Preview
Constructor AppFor
Initialises a new instance of the AppFor class with specified loop parameters.
Syntax
C#
    public AppFor(int from, int to, int step, Action loopBlock)
Parameters
from
The starting value of the loop.
to
The ending value of the loop.
step
The step value (increment or decrement) for each iteration.
loopBlock
The block of code to execute in each iteration of the loop.
Remarks
The loop will iterate from the from value to the to value, and it will increment or decrement based on the step value. The loopBlock will be executed in each iteration.

Method Execute

    public override void Execute()
Exceptions
Exception Condition
Exception Thrown if the loop exceeds the maximum allowed execution cycles, or if the step value is zero or results in an invalid loop direction (infinite loop).
Remarks
The loop will iterate from from to to with the specified step value. A check is performed to ensure the loop does not exceed the maximum allowed execution cycles to avoid infinite loops.
