Class ExportCanvas
The ExportCanvas class provides functionality to export the current state of a canvas as an image file in the specified format (e.g., PNG).
Syntax
C#
    public class ExportCanvas

Documentation Preview
Field canvas
The canvas object that contains the graphical content to export.
Syntax
C#
    private ICanvas canvas

Documentation Preview
Constructor ExportCanvas
Initializes a new instance of the ExportCanvas class with the specified canvas.
Syntax
C#
    public ExportCanvas(ICanvas canvas)
Parameters
canvas
The canvas to be exported. The canvas must implement the ICanvas interface and provide a valid bitmap.

Documentation Preview
Method ExportToImage
Exports the current state of the canvas to an image file at the specified file path.
Syntax
C#
    public void ExportToImage(string filePath)
Parameters
filePath
The full file path (including file name and extension) where the image will be saved. The file format is PNG.
Exceptions
Exception Condition
Exception Thrown when there is an error during the export process (e.g., invalid file path or insufficient permissions).
Remarks
This method retrieves the current bitmap representation of the canvas using canvas.getBitmap(). If the bitmap is valid, it saves the image as a PNG file to the specified location.
Examples
C#
ICanvas myCanvas = new AppCanvas();
ExportCanvas exporter = new ExportCanvas(myCanvas);
exporter.ExportToImage("C:\\Exports\\canvas_output.png");
